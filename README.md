Task 4: Firewall Setup and Configuration

Objective

To configure firewall rules to block and allow specific network ports using Windows Defender Firewall.

---

Tools Used

* Windows Defender Firewall

---

 Steps Performed

1. Opened Windows Defender Firewall
2. Accessed Advanced Settings
3. Viewed existing Inbound Rules
4. Created a rule to block Port 23 (Telnet)
5. Created a rule to allow Port 22 (SSH)
6. Verified rules in Inbound Rules section
7. Deleted the test rule (Block Port 23)

---

Screenshots Included

* Inbound Rules list
* Block Port 23 rule
* Allow Port 22 rule

---

 Result

Firewall rules were successfully configured to block and allow specific ports.

 Conclusion
